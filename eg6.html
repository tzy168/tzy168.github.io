在这个例子中，父元素.parent的边框将不会包围浮动元素，因为父元素的高度没有被浮动元素撑开。

清除浮动的方法：
使用clear属性：可以在一个元素上设置clear: both;来清除左右两侧的浮动。
通常可以在父元素的最后添加一个空的

元素，并设置这个元素的clear属性。


.parent { border: 1px solid black; overflow: auto;// }
.float-element { float:right; width: 100px; height: 100px; background-color: lightblue; }
.clearfix::after { content: ""; display: table; clear: both; }
